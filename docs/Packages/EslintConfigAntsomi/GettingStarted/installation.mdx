## Install dependencies (and peer dependencies)

```bash
npx install-peerdeps --dev @antscorp/eslint-config-antsomi
```

## Configure Eslint

Within your ESLint config file:

```diff
extends: [
+ '@antscorp/antsomi'
]
```

## Configure the ESLint TypeScript parser


This config requires knowledge of your TypeScript config.

In your ESLint config, set [parserOptions.project](https://github.com/typescript-eslint/typescript-eslint/tree/master/packages/parser#parseroptionsproject) to the path of your `tsconfig.json`.

For example:

```diff
{
  extends: ['@antscorp/antsomi'],
+ parserOptions: {
+   project: './tsconfig.json'
+ }
}
```

## Run ESLint

Open a terminal to the root of your project, and run the following command:

```
yarn run eslint --ext js,ts,tsx
```
