Get History of Customer

GET http://api.ants.tech/access/api/event/history

Security​

This api use Token get from Oauth2 as an authentication method.

  • Header: Authorization: Bearer Token
  • In: header

Request Parameters​

NameLocated inDescriptionRequiredTypeExample
customer_idURLCustomer Id in PortalRequiredstring123023902
limitURLQuantity of event you want to fetchOptionalnumber10
event_filtersURLList event filter in responseOptionalstring[{ %22event_action_id%22:%20-103,%20%22event_category_id%22:%20-11}]

Response​

CodeDescription
200Return a list of events
400Bad Request
401Unauthorized.

var headers = new Headers();
headers.append("X-CSCAPI-KEY", "API_KEY");

var requestOptions = {
   method: 'GET',
   headers: headers,
   redirect: 'follow'
};

fetch("https://api.countrystatecity.in/v1/countries", requestOptions)
.then(response => response.text())
.then(result => console.log(result))
.catch(error => console.log('error', error));

var headers = {

http.StreamedResponse response = await request.send();

} else {
  print(response.reasonPhrase);
}
